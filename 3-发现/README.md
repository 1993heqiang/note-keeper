# 👀 发现清单

> 这是一个发现目录，不是待办清单。没有人能评估、学会、用完 100+ 个工具。
>
> 目标是：当你面对一个具体问题时，能回忆起「好像有个工具能做这个」，然后来这里找到它。

## 收录标准

添加前过一遍：

1. **我装了/用了？** → 不应该在这里，去 `1-工具箱/`
2. **我自己写了？** → 不应该在这里，去 `2-笔记/`
3. **解决我当前某个具体问题？** → ✅ 收录，注明解决什么问题
4. **同类已有 3 个？** → 替换最弱的，或直接跳过
5. **「万一以后有用」？** → 浏览器书签，别进仓库

## 维护频率

- 每月整理根目录 README 中的收件箱，分类到本目录
- 每季度检查一次，同类工具超过 5 个时做精简
