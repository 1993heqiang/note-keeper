# 中间件

> 服务器监控、容器、数据库、存储等基础设施工具。

## 监控与运维

| 名称 | 描述 | 地址 |
|---|---|---|
| `Beszel` | 轻量级服务器监控平台，含 Docker 统计和告警。 | [Link](https://github.com/henrygd/beszel) |
| `uptime-kuma` | 精美的自托管监控工具。 | [Link](https://github.com/louislam/uptime-kuma) |
| `sampler` | Shell 命令执行、可视化和告警工具。 | [Link](https://github.com/sqshq/sampler) |
| `sniffnet` | 互联网流量监控工具。 | [Link](https://github.com/GyulyVGC/sniffnet) |
| `1Panel` | 现代化开源 Linux 服务器运维管理面板。 | [Link](https://1panel.cn/) |
| `fastfetch` | 系统信息展示（漂亮格式）。 | [Link](https://github.com/fastfetch-cli/fastfetch) |
