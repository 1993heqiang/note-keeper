# Web 与 API 工具

> Web 开发相关的测试、调试和辅助工具。

## 基准测试

| 名称 | 描述 | 地址 | 推荐指数 |
|---|---|---|---|
| `oha` | HTTP 负载生成器，带 TUI 动画（推荐）。 | [Link](https://github.com/hatoo/oha) | :star::star::star: |
| `hey` | HTTP 负载生成器，ApacheBench 替代品。 | [Link](https://github.com/rakyll/hey) | :star::star: |
| `wrk` | HTTP 性能测试工具。 | [Link](https://github.com/wg/wrk) | :x: |
| `ab` | Apache Bench，Apache HTTP 服务器基准测试工具。 | [Link](https://httpd.apache.org/docs/2.4/programs/ab.html) | :x: |
| `hyperfine` | 命令行基准测试工具。 | [Link](https://github.com/sharkdp/hyperfine) | 待评估 |

## 自动化测试

| 名称 | 描述 | 地址 |
|---|---|---|
| `Reqable` | 新一代 API 调试 + API 测试一站化解决方案（类似 Apifox/Postman）。 | [Link](https://reqable.com/zh-CN/) |
| `bruno` | 创新的开源 API 客户端，Postman 替代品。 | [Link](https://github.com/usebruno/bruno) |
| `playwright` | Microsoft 出品，自动化浏览器工具，支持 Chrome/Firefox/Safari/Edge。 | [Link](https://github.com/microsoft/playwright) |
