## 简介

> MCP（Model Context Protocol）是 Anthropic 推出的开放协议，让 AI 模型通过标准化接口连接外部工具和数据源。类似于 AI 的「USB 协议」。

| 名称             | 描述                                                                                         | 地址                                                |
| ---------------- | -------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| `bb-browser`     | 基于真实 Chrome 的 CLI + MCP 服务，复用登录态操作网页，内置 36+ 平台适配器，返回结构化数据。 | [Link](https://github.com/epiral/bb-browser)        |
| `blender-mcp`    | 通过 MCP 协议连接 Blender，AI 直接操控 3D 建模。                                             | [Link](https://github.com/ahujasid/blender-mcp)     |
| `firecrawl`      | 专为 AI 设计的网页抓取与转换工具，将 URL 转为 Markdown/JSON，用于 RAG、Agent 与知识库构建。  | [Link](https://github.com/firecrawl/firecrawl)      |
| `playwright-mcp` | 微软开源，基于 Playwright 为 LLM 提供浏览器自动化能力，无障碍树精准交互。                    | [Link](https://github.com/microsoft/playwright-mcp) |
