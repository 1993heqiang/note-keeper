# Intellj plugin

| 名称                   | 描述                                     |
| -------------------- | -------------------------------------- |
| `kafkalytic`         | Kafka analysis tool                    |
| `jclasslib`          | java字节码查看器                             |
| `SonarLint`          | 代码问题检查与修复                              |
| `Intellij Scripting` | 可以调试基于JVM的非java语言,例如:Java、Ruby、Python等 |
| `Statistic`          | 代码统计插件                                 |
| `MarsCode`           | MarsCode 是豆包旗下的智能编程助手                  |
| `TONGYI Lingma`      | 通义灵码，是一款基于通义大模型的智能编码辅助工具               |
